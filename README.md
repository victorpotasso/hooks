# Collection of vanilla hooks

## Import

### Web Modules

```
<script type="module">
  import * as hooks from "https://cdn.jsdelivr.net/npm/@victorpotasso/hooks@0.0.1/lib/index.js";
</script>
```

### Node Modules

```npm install --save @victorpotasso/hooks```

```import * as hooks from  "@victorpotasso/hooks"```
